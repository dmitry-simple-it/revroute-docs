# Клиентский скрипт RevRoute: введение

Клиентский скрипт RevRoute — лёгкий скрипт, который помогает:

- сохранить идентификатор атрибуции из параметров ссылки (например, `dub_id`) в first‑party cookie
- отправлять lead/sale события с фронтенда (conversion tracking)
- включать кросс‑доменный трекинг (добавлять идентификатор к исходящим ссылкам на ваши приложения)

## Подключение

Базовая версия скрипта доступна по адресу:

```html
<script defer src="https://app.revroute.ru/analytics/script.js"></script>
```

Доступны варианты скрипта под разные кейсы. Подробности — в разделе “Варианты скрипта”.
