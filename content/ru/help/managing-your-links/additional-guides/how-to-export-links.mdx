---
title: "Как экспортировать ссылки из Dub"
description: "Узнайте, как экспортировать ссылки из вашего рабочего пространства Dub в CSV-файл."
sidebarTitle: "Экспорт ссылок"
---

В рабочем пространстве Dub вы можете экспортировать ссылки в CSV-файл. Это полезно, если вы хотите проанализировать данные локально или поделиться ими с внешними заинтересованными лицами.

Чтобы экспортировать ссылки, выполните следующие шаги:

<Steps>

<Step title="Откройте модальное окно экспорта ссылок">

    В рабочем пространстве Dub перейдите на вкладку **Links**.

    Затем нажмите кнопку `⋮`, чтобы открыть выпадающее меню дополнительных действий. Выберите опцию **Export links**.

    ![Нажмите на выпадающее меню для открытия модального окна экспорта ссылок](https://assets.dub.co/help/links-dashboard-dropdown.png)

</Step>

<Step title="Выберите диапазон дат">

    Выберите диапазон дат для ссылок, которые хотите экспортировать. По умолчанию Dub экспортирует все ссылки.

    ![Выберите диапазон дат для экспортируемых ссылок](https://assets.dub.co/changelog/export-links.png)

</Step>

<Step title="Выберите столбцы для экспорта">

    По умолчанию Dub экспортирует следующие столбцы:
    - Short link
    - Destination URL
    - Clicks
    - Created at

    Вы можете выбрать столбцы для экспорта в разделе **Columns**.

    ![Выберите столбцы для экспорта](https://assets.dub.co/help/export-links-columns.png)

</Step>

<Step title="Примените текущие фильтры" optional>

    При желании вы можете применить текущие фильтры к экспортируемым ссылкам. Если вы это сделаете, Dub экспортирует ссылки, соответствующие текущим фильтрам; в противном случае будут экспортированы все ссылки.

    ![Примените текущие фильтры к экспортируемым ссылкам](https://assets.dub.co/help/export-links-columns-with-filters.png)

</Step>

<Step title="Скачайте данные">

    После выбора диапазона дат и столбцов для экспорта нажмите **Export links**.

    Через несколько секунд CSV-файл будет загружен на ваш компьютер.

</Step>

</Steps>
