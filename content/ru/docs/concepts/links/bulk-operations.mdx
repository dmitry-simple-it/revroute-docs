Revroute позволяет выполнять массовые операции со ссылками. Это особенно полезно, когда необходимо [создать](/docs/api-reference/links/bulk-create), [обновить](/docs/api-reference/links/bulk-update) или [удалить](/docs/api-reference/links/bulk-delete) несколько ссылок одновременно без выполнения множества API-запросов.

## Массовое создание ссылок

Массовое создание позволяет создавать до 100 ссылок одновременно.

<Warning>
  Массовое создание ссылок не поддерживает [пользовательские превью
  ссылок](/help/article/custom-link-previews). Также [события
  вебхуков](/docs/webhooks/introduction) не будут срабатывать при массовом
  создании ссылок.
</Warning>

<CodeGroup>

```javascript Node.js
import { Revroute } from "dub";

export const dub = new Revroute({
  token: process.env.DUB_API_KEY,
});

const result = await dub.links.createMany([
  {
    url: "https://google.com",
  },
  {
    url: "https://google.uk",
  },
]);
```

```go Go
package main

import(
  "context"
  dubgo "github.com/dubinc/dub-go"
  "github.com/dubinc/dub-go/models/operations"
  "log"
  "os"
)

func main() {
  ctx := context.Background()

  s := dubgo.New(
    dubgo.WithSecurity(os.Getenv("DUB_API_KEY")),
  )
}

res, err := s.Links.CreateMany(ctx, []operations.RequestBody{
  operations.RequestBody{
    URL: "https://google.com",
  },
  operations.RequestBody{
    URL: "https://google.uk",
  },
})
```

```python Python
import os
import dub
from dub.models import operations

d = dub.Revroute(
  token=os.environ['DUB_API_KEY'],
)

res = dub.links.create_many(request=[
  {
     "url": "https://google.com",
  },
  {
    "url": "https://google.uk",
  },
])
```

```ruby Ruby
require 'dub'

s = ::OpenApiSDK::Revroute.new
s.config_security(
  ::OpenApiSDK::Shared::Security.new(
    token: "DUB_API_KEY",
  )
)

req = [
  ::OpenApiSDK::Operations::RequestBody.new(
    url: "https://google.com",
  ),
  ::OpenApiSDK::Operations::RequestBody.new(
    url: "https://example.uk"
  ),
]

res = s.links.create_many(req)
```

```php PHP
declare(strict_types=1);

require 'vendor/autoload.php';

use Revroute;
use Revroute\Models\Operations;

$sdk = Revroute\Revroute::builder()->setSecurity('DUB_API_KEY')->build();

$request = [
  new Operations\RequestBody(
    url: 'https://google.com',
  ),
  new Operations\RequestBody(
    url: 'https://google.uk',
  ),
];

$response = $sdk->links->createMany(
  request: $request
);
```

```bash cURL
curl --request POST \
  --url https://api.revroute.ru/links/bulk \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json'
  --data '[
    {
      "url": "https://google.com"
    },
    {
      "url": "https://google.uk"
    }
  ]'
```

</CodeGroup>

Ознакомьтесь с [полным справочником API для эндпоинта массового создания ссылок](/docs/api-reference/links/bulk-create).

## Массовое обновление ссылок

Массовое обновление позволяет изменять до 100 ссылок одновременно **с одинаковыми данными**.

Возможные сценарии использования:

- Присвоение тегов нескольким ссылкам одновременно
- Установка одинакового срока действия для нескольких ссылок
- Обновление UTM-параметров для нескольких ссылок

<Warning>
  С помощью этого эндпоинта нельзя обновить домен или ключ ссылки. Также
  [события вебхуков](/docs/webhooks/introduction) не будут срабатывать при
  массовом обновлении ссылок
</Warning>

<CodeGroup>

```javascript Node.js
import { Revroute } from "dub";

export const dub = new Revroute({
  token: process.env.DUB_API_KEY,
});

const result = await dub.links.updateMany({
  linkIds: ["clux0rgak00011...", "clux0rgak00022..."],
  data: {
    utm_source: "facebook",
    utm_medium: "cpc",
  },
});
```

```go Go
package main

import(
  "context"
  dubgo "github.com/dubinc/dub-go"
  "github.com/dubinc/dub-go/models/operations"
  "log"
  "os"
)

func main() {
  ctx := context.Background()

  s := dubgo.New(
    dubgo.WithSecurity(os.Getenv("DUB_API_KEY")),
  )

  res, err := s.Links.UpdateMany(ctx, operations.BulkUpdateLinksRequestBody{
    LinkIds: []string{
      "clux0rgak00011...",
      "clux0rgak00022...",
    },
    Data: map[string]string{
      "utm_source": "facebook",
      "utm_medium": "cpc",
    },
  })
}
```

```python Python
import os
import dub
from dub.models import operations

d = dub.Revroute(
  token=os.environ['DUB_API_KEY'],
)

res = dub.links.update_many(request={
  "link_ids": [
    "clux0rgak00011...",
    "clux0rgak00022...",
  ],
  "data": {
    "utm_source": "facebook",
    "utm_medium": "cpc",
  },
})
```

```ruby Ruby
require 'dub'

s = ::OpenApiSDK::Revroute.new
s.config_security(
  ::OpenApiSDK::Shared::Security.new(
    token: "DUB_API_KEY",
  )
)

req = ::OpenApiSDK::Operations::BulkUpdateLinksRequestBody.new(
  link_ids: [
    "clux0rgak00011...",
    "clux0rgak00022...",
  ],
  data: {
    "utm_source": "facebook",
    "utm_medium": "cpc",
  },
)

res = s.links.update_many(req)
```

```php PHP
declare(strict_types=1);

require 'vendor/autoload.php';

use Revroute;
use Revroute\Models\Operations;

$sdk = Revroute\Revroute::builder()->setSecurity('DUB_API_KEY')->build();

$response = $sdk->links->updateMany(
  linkIds: [
    'clux0rgak00011...',
    'clux0rgak00022...',
  ],
  data: {
    "utm_source": "facebook",
    "utm_medium": "cpc",
  },
);
```

```bash cURL
curl --request PATCH \
  --url https://api.revroute.ru/links/bulk \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{"link_ids": ["clux0rgak00011...", "clux0rgak00022..."], "data": {"utm_source": "facebook", "utm_medium": "cpc"}}'
```

</CodeGroup>

Ознакомьтесь с [полным справочником API для эндпоинта массового обновления ссылок](/docs/api-reference/links/bulk-update).

## Массовое удаление ссылок

С помощью массового удаления можно удалить до 100 ссылок одновременно.

<Warning>
  Это деструктивное действие, которое невозможно отменить. Действуйте с осторожностью.
  Также [события вебхуков](/docs/webhooks/introduction) не будут срабатывать при
  использовании этого эндпоинта.
</Warning>

<CodeGroup>

```javascript Node.js
import { Revroute } from "dub";

export const dub = new Revroute({
  token: process.env.DUB_API_KEY,
});

const result = await dub.links.deleteMany({
  linkIds: ["clux0rgak00011...", "clux0rgak00022..."],
});
```

}
