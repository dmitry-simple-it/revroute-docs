## Где найти партнеров программы

В меню **Partner Program** нажмите **All Partners**, чтобы увидеть всех участников вашей программы.

![Расположение партнеров программы](https://assets.dub.co/cms/program-partners-location.png)
## Приглашение партнеров

На странице **Partners** нажмите **Invite partner**, чтобы открыть панель приглашения.

![Приглашение партнеров](https://assets.dub.co/cms/invite-partner-button.png)
[Подробнее о приглашении партнеров.](/help/managing-a-program/recruiting-partners/inviting-partners)

## Импорт партнеров

Нажмите меню `⋮`, чтобы импортировать партнеров с поддерживаемых платформ в вашу программу Revroute.

![Импорт партнеров](https://assets.dub.co/cms/import-partners-dropdown.png)

Revroute в настоящее время поддерживает импорт из:

- [Rewardful](/help/managing-a-program/migration-guides/migrating-from-rewardful)
- [Tolt](/help/managing-a-program/migration-guides/migrating-from-tolt)
- [PartnerStack](/help/managing-a-program/migration-guides/migrating-from-partnerstack)
- [FirstPromoter](/help/managing-a-program/migration-guides/migrating-from-firstpromoter)

Если вы хотите импортировать с других платформ, [сообщите нам](https://dub.co/contact/support).

## Экспорт партнеров

Нажмите меню `⋮` и выберите **Export as CSV**, чтобы скачать данные о партнерах для использования в любом месте.

![Выпадающее меню партнеров](https://assets.dub.co/cms/export-partners-dropdown.png)

## Таблица партнеров

Таблица партнеров дает вам полное представление обо всех участниках вашей программы. По умолчанию партнеры отсортированы по **чистой выручке** (net revenue), начиная с наиболее результативных. Это позволяет быстро увидеть, кто приносит наибольшую ценность.

![Таблица партнеров](https://assets.dub.co/cms/partners-table-view.png)

Вы можете сортировать таблицу по любому заголовку столбца. Например, выберите **Conversions**, чтобы отсортировать партнеров по количеству конверсий вместо выручки.

### Столбцы по умолчанию

При первом открытии таблицы вы увидите набор столбцов, предназначенный для быстрой оценки эффективности партнеров.

| Столбец     | Описание                                                  |
| ----------- | --------------------------------------------------------- |
| Group       | Группа, в которой состоит партнер                         |
| Location    | Указанное местоположение партнера                         |
| Clicks      | Общее количество кликов по ссылкам партнера               |
| Leads       | Общее количество лидов, привлеченных ссылками партнера    |
| Conversions | Общее количество лидов, ставших платящими клиентами       |
| Revenue     | Общая выручка от ссылок партнера                          |
| Commissions | Общая сумма комиссий, выплаченных партнеру за рефералов   |
| Net Revenue | Чистая выручка после вычета комиссий                      |

![Таблица партнеров](https://assets.dub.co/cms/partners-table-view-columns.png)

Их можно изменить в любое время, нажав на `значок шестеренки` в правом верхнем углу таблицы, что позволяет включать или отключать дополнительные столбцы.

### Дополнительные столбцы

Если вам нужен более детальный анализ, вы можете включить дополнительные метрики эффективности и атрибуции. Эти дополнительные столбцы помогут лучше понять качество партнеров, эффективность воронки и долгосрочную ценность.

| Столбец       | Описание                                                | Формула                             |
| ------------- | ------------------------------------------------------- | ----------------------------------- |
| EPC           | Доход за клик                                           | Общая выручка / Общее кол-во кликов     |
| Avg LTV       | Средняя пожизненная ценность каждого платящего клиента  | Общая выручка / Общее кол-во конверсий  |
| Click -> Lead | Процент кликов, ставших лидами                          | Общее кол-во лидов / Общее кол-во кликов |
| Click -> Conv | Процент кликов, ставших платящими клиентами              | Общее кол-во конверсий / Общее кол-во кликов |
| Lead -> Conv  | Процент лидов, ставших платящими клиентами               | Общее кол-во конверсий / Общее кол-во лидов |
| ROAS          | Рентабельность рекламных расходов                       | Общая выручка / Общая сумма комиссий    |

### Фильтрация и поиск

Используйте фильтры для сортировки партнеров по группе, статусу или местоположению. Фильтры помогут выявить наиболее эффективных партнеров.

Вы также можете использовать поле поиска для поиска по имени, электронной почте или ссылке. Это удобно, когда нужно найти конкретного партнера.

![Фильтры партнеров](https://assets.dub.co/cms/partner-filters-2.png)

**Фильтр по статусу**

Фильтруйте по статусу, чтобы увидеть, кто активен, а кто нет.

| Статус      | Описание                                                                                                                                                                |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Approved    | Партнер присоединился и может получать вознаграждения                                                                                                                   |
| Invited     | Партнеры, приглашенные в вашу программу, но еще не принявшие приглашение                                                                                                |
| Declined    | Партнеры, отклонившие ваше приглашение                                                                                                                                  |
| Archived    | Партнер удален из программы                                                                                                                                             |
| Banned      | Партнеры, заблокированные в вашей программе                                                                                                                             |
| Deactivated | Партнеры, с которыми вы деактивировали партнерство. Используется для завершения партнерства по причинам, не связанным с нарушениями (например, коммерческим/финансовым). |

### Дополнительное меню

Для каждого партнера нажмите меню ..., чтобы увидеть дополнительные действия.

![Управление партнером](https://assets.dub.co/cms/partner-more-dropdown.png)

Дополнительные действия включают:

- **Одобренные партнеры**: Просмотр комиссий, Архивирование партнера и Блокировка партнера
  Архивирование партнера и Блокировка партнера
- **Приглашенные партнеры**: Повторная отправка приглашения и
  Удаление приглашения
- **Архивированные партнеры**: Просмотр комиссий, Разархивирование партнера и
  Блокировка партнера.
- **Заблокированные партнеры**: Просмотр комиссий, Архивирование партнера и
  Разблокировка партнера.

## Детали партнера

Нажмите на строку партнера, чтобы увидеть его детали, такие как профиль, эффективность, короткие ссылки, выплаты и комиссии.

![Детали партнера](/images/dub-partners/partner-view.png)

### Ссылки

Чтобы увидеть все реферальные ссылки этого партнера, нажмите **Links**. Вы также увидите коды скидок партнера, если они были добавлены. Подробнее о [кодах скидок](/help/managing-a-program/getting-started/dual-sided-incentives#option-2-using-stripe-promo-codes-no-link-required) партнеров.

![Ссылки партнера](/images/dub-partners/partner-links.png)

Если вам нужно создать ссылку для партнера, нажмите **Create link**.

![Создание ссылки партнера](https://assets.dub.co/cms/create-partner-link.png)

### Выплаты

Нажмите **Payouts**, чтобы увидеть все выплаты партнера. Нажмите на любую строку выплаты, чтобы увидеть её детали.

![Выплаты партнера](/images/dub-partners/partner-payouts.png)

### Профиль

Чтобы просмотреть данные профиля партнера, нажмите кнопку **Profile** рядом с его именем в правом столбце.

![Кнопка профиля партнера](/images/dub-partners/partner-profile-button.png)

Откроется панель, где вы можете увидеть такие данные, как страна, биография и подключенные веб-сайты или аккаунты в социальных сетях. Вы также сможете просмотреть исходный ответ партнера на заявку при первом обращении.

![О партнере](/images/dub-partners/partner-profile-view.png)

### Комментарии

Нажмите **Comments**, чтобы просмотреть и добавить комментарии к этому партнеру, которые видны только вашему рабочему пространству. Вы также можете редактировать и удалять свои комментарии в случае ошибки.

![Комментарии к партнеру](/images/dub-partners/partner-comments.png)
### Комиссии

Нажмите **Commissions**, чтобы открыть страницу с комиссиями, отфильтрованными для этого партнера. [Подробнее о комиссиях партнеров](/help/managing-a-program/rewards-and-commissions/partner-commissions).

![Комиссии партнера](https://assets.dub.co/cms/partner-commissions-filter.png)

### Ручная комиссия

Иногда вам может потребоваться создать разовую или повторяющуюся комиссию для учёта пропущенной комиссии или по другим причинам. В Revroute вы можете легко создавать такие комиссии через панель управления, и они будут включены в следующую выплату для этого партнера.

Нажмите **Create commission**, чтобы начать. [Подробнее о ручных комиссиях](/help/managing-a-program/rewards-and-commissions/partner-commissions#creating-manual-commissions).

![Создание комиссии](/images/dub-partners/partner-create-commission.png)

## Расширенные настройки

<Warning>
  Будьте осторожны при изменении любых расширенных настроек партнера.
</Warning>

Чтобы перейти к расширенным настройкам партнера, на странице его профиля нажмите выпадающее меню `⋮` и выберите **Advanced settings**.

![Расширенные настройки](/images/dub-partners/advanced-settings.png)

### Tenant ID

Если вам нужно обновить tenant ID партнера, изменение ID здесь также обновит [tenantId для всех его ссылок](/docs/concepts/links/organization#tenant-id). Это может повлиять на любую пользовательскую интеграцию, которую вы настроили, поэтому меняйте только в том случае, если вы уверены в правильности и необходимости информации.

### Совместное использование данных о клиентах

Включение этой настройки позволяет данному партнеру видеть данные о клиентах по любым лидам или продажам, которые он совершил.

Вместо скрытых адресов электронной почты они увидят полный адрес электронной почты клиента и его имя. Они также смогут перейти к профилю клиента для просмотра истории продаж.

![Просмотр данных клиента](https://assets.dub.co/cms/partner-customer-view.png)

### Игнорирование правил перемещения между группами

Включение этой настройки позволяет данному партнеру оставаться в установленной группе и игнорировать любые [правила перемещения между группами](/help/managing-a-program/getting-started/partner-groups#moving-partners-between-groups), которые могут к нему применяться.
