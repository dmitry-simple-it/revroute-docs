# Отслеживание конверсий

Conversion tracking позволяет отправлять lead/sale события с фронтенда и связывать их с исходным источником (атрибуцией).

## Требования

- Подключён клиентский скрипт RevRoute
- В настройках воркспейса создан **publishable key** (Settings → Analytics)

## Подключение варианта скрипта

```html
<script defer src="https://app.revroute.ru/analytics/script.conversion-tracking.js"></script>
```

## Отправка событий

Обычно события отправляются через SDK/утилиты в вашем приложении. Конкретные эндпоинты и payload зависят от вашей интеграции.
