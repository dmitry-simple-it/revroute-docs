Short.io — популярный сервис сокращения ссылок, который позволяет создавать короткие ссылки для использования в маркетинговых кампаниях, публикациях в социальных сетях и многом другом.

## Как перейти с Short.io на Revroute

Если вы хотите перейти с Short.io на Revroute, мы создали специальный инструмент миграции, который поможет перенести ваши ссылки за несколько простых шагов.

<Steps>

<Step title="Получите API-ключ Short.io">

Сначала войдите в свой аккаунт Short.io и перейдите на страницу [Integrations & API](https://app.short.io/settings/integrations/api-key).

![](https://assets.dub.co/help/short-api-page.png)

Нажмите "Create API Key" — откроется модальное окно. Оставьте все параметры по умолчанию и нажмите "Create".

![API key creation modal in Short.io](https://assets.dub.co/help/short-create-api-key.png)

Скопируйте созданный API-ключ. Он понадобится на следующем шаге.

![Copy the created API key](https://assets.dub.co/help/short-copy-api-key.png)

</Step>

<Step title="Используйте инструмент миграции Short.io">

Если вы ещё этого не сделали, [создайте рабочее пространство в Revroute](/help/managing-your-workspace/what-is-a-workspace#how-to-create-a-workspace).

<Info>
  **Совет**: После создания рабочего пространства мы рекомендуем **_не
  [настраивать домен](/help/managing-your-links/custom-domains/how-to-add-custom-domain)_** до тех пор, пока
  вы не перенесёте все ссылки. Это предотвратит возможные перебои в работе
  ваших ссылок.
</Info>

Затем на панели управления рабочего пространства нажмите кнопку `⋮` рядом с кнопкой **Create Link** и выберите **Import from Short.io**.

![Import from Short.io button](https://assets.dub.co/help/links-dashboard-dropdown.png)

Вам будет предложено ввести API-ключ Short.io. Вставьте ключ, скопированный на предыдущем шаге, и нажмите **Confirm API Key**.

![Import from Short.io modal](https://assets.dub.co/help/import-from-short-modal.png)

В фоновом режиме мы получим все ваши пользовательские домены Short.io и количество ссылок для каждого из них. Вам будет представлен список найденных доменов.

![Domains found in Short.io and their respective number of links](https://assets.dub.co/help/imported-short-domains.png)

</Step>

<Step title="Выберите домены для миграции">

Выберите домены, которые хотите перенести, и нажмите **Confirm import**. Через несколько секунд все ваши ссылки будут перенесены в Revroute.

![Links imported from Short.io](https://assets.dub.co/help/imported-shortio-links.png)

После импорта всех ссылок мы отправим вам электронное письмо с уведомлением о завершении миграции.

![Email notification about completed Short.io migration](https://assets.dub.co/help/imported-short-email.png)

</Step>

<Step title="Настройте домен">

После получения письма вы можете [настроить домен](/help/managing-your-links/custom-domains/how-to-add-custom-domain#step-2-configure-your-domain), указав правильные DNS-записи. Это обеспечит, что ваши ссылки будут указывать на Revroute, а не на Short.io. После этого вы можете начать использовать ваши ссылки!

</Step>

</Steps>

## Какие атрибуты ссылок Short.io переносятся?

Вот список атрибутов, которые переносятся из Short.io в Revroute:

1. URL назначения
2. Слаг короткой ссылки
3. Дата создания ссылки
4. Таргетинг на устройства iOS (если включён)
5. Таргетинг на устройства Android (если включён)
6. Статус архивации
7. Заголовок ссылки (в рамках функции [пользовательских превью ссылок](/help/article/custom-link-previews) в Revroute)
8. [Теги ссылок](/help/managing-your-links/links/how-to-use-tags)

Вот атрибуты, которые не переносятся:

1. Защита паролем
2. URL для ошибки 404
3. Дата истечения срока действия ссылки
4. Тип редиректа ссылки (Revroute использует редиректы 307 для всех ссылок)
5. Маскировка ссылок

Если вы хотите, чтобы переносились другие атрибуты, [напишите нам](mailto:support@dub.co?subject=Short.io%20migration%20feedback), и мы постараемся это реализовать!
