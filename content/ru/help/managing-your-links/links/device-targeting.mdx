<Tip>
  Эта функция доступна только на [тарифных планах Pro и
  выше](https://dub.co/pricing).
</Tip>

Таргетинг по устройствам — это мощная функция, которая позволяет персонализировать целевой URL ваших ссылок в зависимости от типа устройства пользователя.

Например, вы можете установить пользовательский целевой URL для устройств iOS, включив переключатель «iOS Targeting» в [конструкторе ссылок](/help/managing-your-links/additional-guides/dub-link-builder), и другой URL для устройств Android, включив переключатель «Android Targeting».

Чтобы использовать эту функцию, нажмите кнопку **Targeting** в конструкторе ссылок. Откроется новое модальное окно с опциями таргетинга по устройствам.

Вы также можете быстро получить доступ к функции Targeting с помощью сочетания клавиш `G` в конструкторе ссылок.

![Использование таргетинга по устройствам для персонализации ссылки](https://assets.dub.co/help/device-targeting.png)

Это особенно полезно, если вы хотите направить пользователей iOS в App Store для скачивания вашего iOS-приложения, а пользователей Android — в Google Play Store для скачивания вашего Android-приложения.

В обоих случаях основной **Destination URL** будет использоваться как резервный URL, который применяется, если тип устройства пользователя не соответствует правилам таргетинга.
