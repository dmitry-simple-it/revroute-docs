---
title: "Как фильтровать аналитику по тегам"
description: "Узнайте, как создавать мощные отчёты по кампаниям для ваших ссылок с помощью функции тегов."
sidebarTitle: "По тегам"
---

В Revroute вы можете фильтровать аналитику по [тегам](/help/managing-your-links/links/how-to-use-tags). Это позволяет создавать мощные отчёты по кампаниям для ваших ссылок.

Например, вы проводите рекламную кампанию с инфлюенсерами в Instagram с несколькими ссылками и хотите увидеть, как они работают.

Вы можете сгруппировать ссылки с тегом **Instagram Influencers**, а затем отфильтровать аналитику по этому тегу, чтобы получить комплексный отчёт об их эффективности.

---

Прежде чем вы сможете фильтровать аналитику по тегам, вам нужно [создать тег](/help/managing-your-links/links/how-to-use-tags#how-to-create-a-tag) и [назначить его ссылкам, по которым хотите фильтровать](/help/managing-your-links/links/how-to-use-tags#can-i-assign-multiple-tags-to-a-link).

<CheckList>
  <CheckListItem
    title="Создайте тег"
    href="/help/managing-your-links/links/how-to-use-tags#how-to-create-a-tag"
  >
    Если вы ещё этого не сделали, создайте тег для фильтрации аналитики.
  </CheckListItem>
  <CheckListItem
    title="Назначьте тег соответствующим ссылкам"
    href="/help/managing-your-links/links/how-to-use-tags#can-i-assign-multiple-tags-to-a-link"
  >
    Добавьте тег ко всем ссылкам, по которым хотите фильтровать.
  </CheckListItem>
</CheckList>

Затем, когда ваши ссылки начнут накапливать клики, вы сможете фильтровать аналитику по тегам.

<Steps>

     <Step title="Откройте выпадающий фильтр аналитики">
        На вкладке [Analytics](https://app.revroute.ru/analytics) вашего рабочего пространства Revroute нажмите на кнопку **Filter**, чтобы открыть выпадающий фильтр:

        ![Выпадающий фильтр аналитики](https://assets.dub.co/help/analytics-filter-dropdown.png)

        <Tip>
          Вы также можете использовать сочетание клавиш `F` для быстрого открытия выпадающего фильтра.
        </Tip>
    </Step>
    <Step title="Выберите фильтр 'Tags'">
        Выберите опцию **Tags** в выпадающем меню, чтобы открыть фильтр по тегам:

![Фильтр аналитики по тегам](https://assets.dub.co/help/analytics-tags-dropdown.png)

    </Step>
    <Step title="Нажмите на тег, по которому хотите фильтровать">
        Наконец, нажмите на тег, по которому хотите фильтровать:

        ![Отфильтрованная аналитика по тегам](https://assets.dub.co/help/filtered-analytics-by-tags.png)
    </Step>

    <Step title="Необязательно: фильтрация по нескольким тегам">
        Вы также можете фильтровать по нескольким тегам, выбрав несколько тегов в выпадающем меню.

        ![Отфильтрованная аналитика по нескольким тегам](https://assets.dub.co/help/filtered-analytics-by-multiple-tags.png)
    </Step>

</Steps>

Готово! Вы успешно отфильтровали аналитику по тегам. Вы также можете [экспортировать данные аналитики](/help/managing-your-links/analytics/how-to-export-analytics) в CSV-файл для дальнейшего анализа.
