<Tip>
  Эта функция доступна только на [тарифных планах Pro и
  выше](https://dub.co/pricing).
</Tip>

По умолчанию, когда пользователь переходит по короткой ссылке, которая не существует в Dub, он увидит страницу-заглушку «Link Not Found»:

![Страница 404-заглушка в Dub](https://assets.dub.co/changelog/not-found-support.jpg)

Если вы [используете пользовательский домен в Dub](/help/managing-your-links/custom-domains/how-to-add-custom-domain), вы можете задать пользовательский URL для перенаправления пользователей, когда короткая ссылка в вашем домене не найдена.

Чтобы настроить пользовательский URL для страницы «не найдено» для ваших доменов, выполните следующие шаги:

1. Перейдите на [вкладку Domains настроек рабочего пространства](https://app.dub.co/domains).
2. Нажмите кнопку `⋮` рядом с доменом, для которого хотите настроить пользовательский URL «не найдено».
3. Выберите **Edit Domain** для редактирования настроек домена.
4. Активируйте переключатель секции **Not Found URL** и введите пользовательский URL для перенаправления пользователей, когда короткая ссылка в вашем домене не найдена.
5. Нажмите кнопку **Save changes** для сохранения изменений.

![Настройка URL для «не найдено» для пользовательского домена](https://assets.dub.co/help/set-not-found-url-for-domain.png)
