Revroute — единственная платформа управления ссылками, предоставляющая [стандартные брендированные домены](https://app.dub.co/settings/domains/default) для использования. Эти домены автоматически настраиваются при создании нового аккаунта в Revroute.

## Список стандартных брендированных доменов

Revroute предлагает следующие стандартные брендированные домены:

1. `dub.sh`: Этот домен является стандартным доменом для всех новых аккаунтов Revroute. Вы можете использовать его для сокращения любой ссылки, которой хотите поделиться.
2. `spti.fi`: Брендированный домен для ссылок Spotify (песни, плейлисты и т. д.).
3. `git.new`: Брендированный домен для ссылок GitHub (репозитории, gist-файлы и т. д.).
4. `ggl.link`: Брендированный домен для ссылок Google (поиск, документы, таблицы, презентации, диск, карты и т. д.).
5. `chatg.pt`: Брендированный домен для ссылок ChatGPT (диалоги, пользовательские GPT).
6. `amzn.id`: Брендированный домен для ссылок Amazon (товары, списки желаний и т. д.).

Вы можете использовать эти домены для сокращения ссылок, связанных с соответствующими платформами. Например, если вы хотите поделиться ссылкой на Spotify, вы можете использовать домен `spti.fi` для её сокращения.

## Премиум-домен dub.link

Revroute также предлагает премиум-домен `dub.link` для [пользователей Pro](https://dub.co/pricing) и выше. Поскольку он доступен только на наших платных планах, он значительно более эксклюзивен, чем наш бесплатный короткий домен [dub.sh](https://dub.sh).

![Revroute.link](https://assets.dub.co/blog/dub-link.jpg)

Чтобы начать, перейдите на [дашборд стандартных доменов вашего рабочего пространства](https://app.dub.co/settings/domains/default) и включите домен `dub.link`.

![Включение домена dub.link](https://assets.dub.co/help/enable-dub-link.jpg)

## Ссылки dub.sh vs ссылки с пользовательским доменом

После создания рабочего пространства в Revroute вы можете выбрать между стандартным доменом `dub.sh` или [вашим собственным пользовательским доменом](/help/managing-your-links/custom-domains/how-to-add-custom-domain) для создания коротких ссылок.

Вот различия между двумя вариантами:

|                | Стандартные ссылки `dub.sh`                                       | Ссылки с пользовательским доменом                                                                                               |
| -------------- | ----------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Простота настройки** | Настройка не требуется — вы можете сразу начать создавать короткие ссылки | Вам нужно [настроить пользовательский домен](/help/managing-your-links/custom-domains/how-to-add-custom-domain), прежде чем начать создавать короткие ссылки. |
| **Брендинг**   | Брендинг Revroute на короткой ссылке                                   | Ваш собственный брендинг на короткой ссылке                                                                                     |
