# Tone of voice (RU, B2B)

- Обращение на «вы» с маленькой буквы в середине предложения.
- Активный залог: «Настройте», «Запустите», «Отслеживайте» вместо пассива «может быть настроено».
- Избегать кальки с английского («менеджить партнёрство», «инбокс решений») — перефразировать по-русски.
- Термины из glossary.json соблюдать; не смешивать «аффилиат» и «партнёр» без причины — по умолчанию «партнёр».
- Числа и валюты: пробел перед ₽/€/$ по типографике (`10 000 ₽`); диапазоны через en-dash по желанию.
- Кавычки: «ёлочки» для выделения смысла; названия продуктов без кавычек, если это бренд.
- Не раздувать заголовки: короче и яснее; подзаголовок раскрывает обещание заголовка.
