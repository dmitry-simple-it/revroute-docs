## Шаг 2: Установите приложение Revroute для Shopify

<Steps>

<Step title="Установите приложение Shopify">
  Установите [приложение Revroute для Shopify](https://d.to/shopify/app) из App Store.

  ![The connection status in the Revroute app](/images/shopify/shopify-app.png)

</Step>

<Step title="Подключите Shopify к рабочему пространству Revroute">
  После установки вам будет предложено связать одно из рабочих пространств Revroute
  с приложением. Нажмите **Connect**, чтобы установить соединение между
  вашим магазином Shopify и рабочим пространством Revroute.

После этого шага вы будете перенаправлены обратно в ваш магазин Shopify и увидите список ссылок из вашего рабочего пространства Revroute:

![The list of links in your Revroute workspace](/images/shopify/shopify-links-table.png)

С помощью приложения Shopify вы также можете создавать [ссылки с включённым отслеживанием конверсий](/docs/conversions//quickstart#step-1-enable-conversion-tracking-for-your-links) непосредственно из вашего магазина Shopify:

![Create a conversion-enabled link from your Shopify store](/images/shopify/shopify-create-link.png)

Для более мощного создания ссылок вы также можете использовать [конструктор ссылок Revroute](/help/managing-your-links/additional-guides/dub-link-builder).

</Step>

<Step title="Активируйте скрипт Revroute Analytics">
  После установки приложения Revroute для Shopify скрипт Revroute Analytics добавляется как встраиваемый компонент приложения. Однако его необходимо активировать вручную, чтобы он был включён в вашу текущую тему.

Чтобы активировать скрипт Revroute Analytics, выполните следующие шаги:

1. Перейдите в панель администрирования Shopify.
2. Откройте **Online Store** > **Themes**.
3. Нажмите **Customize** для вашей текущей темы.
4. В редакторе темы выберите вкладку **App embeds**.
5. Найдите **Analytics Script** для приложения Revroute Shopify и включите его.

![Enable the Revroute Analytics script in your Shopify theme](/images/shopify/shopify-enable-tracking-script.png)

</Step>

</Steps>

<Tip>

Интеграция Revroute с Shopify автоматически перенаправляет следующие события в Revroute:

- `orders/paid`: это событие срабатывает, когда клиент совершает покупку в вашем магазине Shopify. Оно используется для отслеживания продаж, пришедших по ссылкам Revroute.
- `app/uninstalled`: это событие происходит при удалении приложения из магазина. Оно используется для удаления интеграции из вашего рабочего пространства Revroute.

Кроме того, мы также подписываемся на обязательные вебхуки соответствия, требуемые Shopify.

</Tip>

## Поддержка конвертации валют

Для простоты Revroute записывает все продажи в базовой валюте магазина Shopify. Например, если ваш магазин Shopify использует USD, Revroute будет записывать все продажи в USD — даже если ваши клиенты платят в другой валюте.

```json orders/paid
// Shopify orders/paid event payload
// @see: https://shopify.dev/docs/api/webhooks?reference=toml#list-of-topics-orders/paid
{
  ...
  "current_subtotal_price_set": {
    "shop_money": {
      "amount": "398.00", // this is the amount that Revroute will record
      "currency_code": "USD" // this is the currency of your Shopify store
    },
    "presentment_money": {
      "amount": "572.25",
      "currency_code": "CAD"
    }
  },
  ...
}
```
