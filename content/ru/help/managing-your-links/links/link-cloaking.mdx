С помощью Revroute вы можете скрыть URL назначения за вашей короткой ссылкой.

Чтобы включить маскировку ссылок, нажмите на кнопку дополнительных параметров `...` в конструкторе ссылок и выберите опцию **Add Link Cloaking**.

![Toggling the ](https://assets.dub.co/help/link-cloaking.png)

Это полезно, когда вы хотите показывать ваш бренд или пользовательский домен вместо фактического URL назначения.

Когда эта функция включена, в адресной строке браузера пользователей будет отображаться ваша короткая ссылка вместо URL назначения.

![Masking ](https://assets.dub.co/changelog/link-cloaking.png)

## Ограничения маскировки ссылок

Несколько ограничений функции маскировки ссылок:

1. Для корректной работы маскировки убедитесь, что URL назначения использует протокол `https`. Если URL назначения использует `http`, браузер покажет предупреждение "Not Secure", и маскировка работать не будет.
2. Маскировка ссылок может не работать для некоторых сайтов, которые имеют защитные меры, предотвращающие встраивание:

- Заголовок `X-Frame-Options` установлен в `DENY`
- Заголовок `content-security-policy` установлен в `frame-ancestors 'none'`

## Маскировка ссылок с заголовками безопасности

Если вы контролируете URL назначения, который маскируете, вы можете использовать заголовки безопасности, чтобы включить маскировку ссылок в Revroute и при этом запретить встраивание через iframe на других сайтах.

### Добавление заголовков безопасности к URL назначения

Для этого необходимо добавить ваш [короткий домен Revroute](/help/managing-your-links/custom-domains/how-to-add-custom-domain) в список разрешённых источников на вашем сайте, указав следующие заголовки ответа:

```bash
Content-Security-Policy: frame-ancestors 'self' [shortdomain.com]
```

Например, если ваш короткий домен Revroute — `dub.link`, необходимо добавить следующие заголовки к вашему URL назначения:

```bash
Content-Security-Policy: frame-ancestors 'self' dub.link
```

### Добавление нескольких доменов в белый список

Вы также можете добавить несколько доменов, разделяя их пробелом:

```bash
Content-Security-Policy: frame-ancestors 'self' dub.link otherdomain.com
```

Это гарантирует, что ваш сайт не может быть встроен на сторонних сайтах, но при этом может быть замаскирован вашим коротким доменом Revroute.
