В любой день Revroute отслеживает более **5 миллионов событий кликов, [конверсий](/help/article/dub-conversions) и [комиссий](/help/managing-a-program/rewards-and-commissions/partner-commissions).**

При таком объеме отслеживаемых данных важно иметь возможность легко фильтровать и создавать отчеты, помогающие маркетинговым командам принимать обоснованные решения.

В этом руководстве мы покажем, как использовать вкладку аналитики программы для максимального использования данных о событиях и вовлеченности вашей партнерской программы.

## Где найти аналитику программы?

При просмотре навигации партнерской программы нажмите на страницу **Analytics** в группе **Insights**:

![Расположение аналитики программы](https://assets.dub.co/cms/program-analytics-location.png)

## Представления аналитики программы

Панель аналитики вашей программы состоит из нескольких представлений:

1. [Аналитический график](#1-analytics-chart)
2. [Таблица данных партнеров](#2-partner-data-table)
3. [Агрегированные данные по различным параметрам (топ-представления)](#3-aggregated-data-for-different-facets-top-views)

### 1. Аналитический график

Это представление по умолчанию, показывающее количество событий кликов в виде графика временных рядов. Вы также можете выбрать лиды или продажи для просмотра их графика и данных.

![График временных рядов](https://assets.dub.co/cms/program-analytics-timechart-1.png)

**Переключатель данных**

Вы можете изменить отображаемые данные, нажав на переключатель `123 | $` в правом верхнем углу раздела типа данных.

По умолчанию выбран `$`, так как он напрямую связан с вашими [типами вознаграждений](/help/managing-a-program/rewards-and-commissions/partner-rewards). Переключение на `123` изменит любые денежные значения на их количественное представление. Если в вашей программе есть другие денежные вознаграждения (клики, лиды, продажи), их долларовая сумма также будет отображаться.

![График временных рядов](https://assets.dub.co/cms/program-analytics-datatoggle.png)

**Переключатель графика**

Вы можете изменить тип отображаемого графика, нажав на переключатель `time-chart | conversion funnel` в правом верхнем углу области графика.

![График временных рядов](https://assets.dub.co/cms/program-analytics-charttoggle.png)

При переключении типа графика на **воронку конверсий** вы можете увидеть процент клиентов, завершивших каждый этап конверсии продажи. В этом примере 0,58% кликов привели к покупке.

![График временных рядов](https://assets.dub.co/cms/program-analytics-funnel.png)

### 2. Таблица данных партнеров

Эта таблица показывает эффективность ваших партнеров по каждому из типов данных из графика временных рядов (клики, лиды и продажи). Соответствующий столбец данных отсортирован от большего к меньшему, чтобы вы могли быстро увидеть лучшего исполнителя в этой категории.

![Таблица данных партнеров](https://assets.dub.co/cms/program-analytics-partnertable-revenue.png)

### 3. Агрегированные данные по различным параметрам (топ-представления)

Это более известные как «Топ-представления» в Revroute Analytics. Эти представления показывают ваши топ-ссылки, страны, города, устройства и многое другое.

![Агрегированные данные по различным параметрам](https://assets.dub.co/cms/program-analytics-topviews.png)

Каждый раздел имеет собственные вкладки для выбора различных данных для дальнейшего изучения данных вашей программы. Они сгруппированы по следующим категориям:

- **Links** — Короткие ссылки и URL назначения
- **Referrals** — Рефереры и UTM-параметры
- **Location** — Страны, города, регионы и континенты
- **Tech** — Устройства, браузеры, ОС и триггеры

## Фильтрация данных

При таком объеме данных важно иметь возможность фильтровать и принимать лучшие решения. Нажмите кнопку **Filter** в левом верхнем углу, чтобы увидеть доступные варианты.

![Фильтрация аналитики программы](https://assets.dub.co/cms/program-analytics-filter.png)

| Фильтр    | Описание                                                                                             |
| --------- | ---------------------------------------------------------------------------------------------------- |
| Ask AI    | Используйте естественный язык для запросов к аналитике — например: «mobile chrome users US only», «UK android users» |
| Group     | Выберите конкретную группу партнеров                                                                 |
| Partner   | Выберите конкретного партнера программы                                                               |
| Country   | Выберите конкретную страну                                                                           |
| City      | Выберите конкретный город                                                                            |
| Continent | Выберите конкретный континент — N. America, Europe и т.д.                                            |
| Device    | Выберите конкретное устройство — Desktop, Mobile, Tablet и т.д.                                      |
| Browser   | Выберите конкретный браузер — Chrome, Safari, Firefox и т.д.                                         |
| OS        | Выберите конкретную операционную систему — Windows, Mac OS и т.д.                                    |
| Referer   | Выберите конкретный веб-сайт, с которого пришел посетитель                                           |
