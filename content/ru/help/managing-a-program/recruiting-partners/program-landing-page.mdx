## Что такое лендинг программы?

Лендинг программы предоставляет заявителям информацию о вашей программе и предлагаемых вознаграждениях. Он также включает файлы бренда или маркетинговые материалы, разделы FAQ и любой другой контент, который поможет заявителю понять вашу программу, что представляет собой ваш продукт и почему стоит подать заявку.

<Tip>
  В Revroute вы можете создать отдельный лендинг для каждой [группы
  партнёров](/help/managing-a-program/getting-started/partner-groups) в вашей программе. Это удобно для
  демонстрации различной информации о программе, адаптированной для разных
  типов партнёров — например, инфлюенсеров, аффилиатов или реселлеров.

По умолчанию при создании новой группы партнёров используются данные лендинга
из группы партнёров по умолчанию, но вы всегда можете отредактировать их
после создания.

</Tip>

## Где найти конструктор лендинга?

На странице **Groups** откройте группу, для которой хотите настроить лендинг:

![Выбор группы](https://assets.dub.co/cms/program-group-click.png)

В навигации по вкладкам нажмите **Branding**, затем в панели контента выберите **Landing page**, чтобы открыть редактор.

![Страница брендинга в панели управления](https://assets.dub.co/cms/program-branding-tab2.png)

## Редактирование и добавление контента

При наведении курсора на содержимое лендинга в конструкторе вы заметите появление действий, позволяющих добавлять, редактировать, удалять и менять порядок блоков контента на странице.

![Страница бренда в панели управления](https://assets.dub.co/cms/lander-edit1.jpg)

Контент над разделом вознаграждений можно только редактировать. Это сделано для
поддержания единообразия контента. Контент ниже кнопки **«Apply today»** можно добавлять, редактировать,
удалять и менять порядок. ## Типы контента Нажатие на «Insert block» позволяет
выбрать тип контента, который вы хотите добавить.

![Страница бренда в панели управления](https://assets.dub.co/cms/lander-insertblock1.jpg)

### Текст

Лучше всего подходит для текстового и абзацного контента. Этот раздел также использует форматирование markdown при редактировании.

![Добавление текстового контента](https://assets.dub.co/cms/lander-insert-text.jpg)

### Изображение

Для добавления скриншотов продукта и других дополнительных визуальных материалов программы.

![Добавление изображений](https://assets.dub.co/cms/lander-insert-image.jpg)

### Файлы

Для добавления скриншотов продукта и других дополнительных визуальных материалов программы.

![Добавление файлов](https://assets.dub.co/cms/lander-insert-file.jpg)

### Аккордеон

Лучше всего подходит для разделов FAQ и другого сворачиваемого контента.
Этот раздел также использует форматирование markdown при редактировании.

![Добавление аккордеона](https://assets.dub.co/cms/lander-insert-accordion.jpg)

### Калькулятор заработка

Дайте вашим партнёрам представление о том, сколько они могут заработать, на основе ваших вознаграждений по умолчанию.

![Добавление калькулятора заработка](https://assets.dub.co/cms/lander-insert-calculator.jpg)

## Генерация лендинга с помощью ИИ

Вы также можете использовать наш генератор лендингов на базе ИИ для быстрого создания каркаса лендинга для вашей партнёрской программы.

Наш ИИ просканирует ваш веб-сайт, извлечёт всю необходимую информацию и создаст привлекательный лендинг с данными о вашей компании, информацией о ценах, функциях и многом другом.

Чтобы начать, просто нажмите кнопку «Generate» и введите URL вашего веб-сайта (он должен быть уже предзаполнен из настроек программы):

![Кнопка генерации лендинга с помощью ИИ](https://assets.dub.co/cms/generate-lander-ai-button.png)

Вот краткая демонстрация того, как это выглядит:

<video src="https://assets.dub.co/cms/generate-lander-ai-demo.mp4"
    loop
    autoPlay
    muted
    playsInline
  ></video>

<Warning>
  ИИ может допускать ошибки — обязательно проверьте весь сгенерированный контент
  на точность и стиль перед публикацией изменений.
</Warning>

## Публикация изменений

После завершения работы с контентом нажмите кнопку Publish в правом верхнем углу, чтобы опубликовать изменения.

![Кнопка публикации](https://assets.dub.co/cms/lander-publish-2.png)

## Доступные стили Markdown
| Контент       | Пример                  |
| ---------- | ---------------------- |
| Заголовки       | `# H1` , `# H2` , `# H3` |
| Жирный        | `**bold text**`  |
| Курсив   | `*italicized text*` |
| Цитата | `> blockquote` |
| Горизонтальная линия   |  `---` |
| Ссылка | `[title](https://www.example.com)`          |
| Нумерованный список | `1. First item` , `2. Second item` , `3. Third item` |
| Маркированный список | `- First item` , `- Second item` , `- Third item` |
