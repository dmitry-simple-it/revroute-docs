## Что такое форма заявки на участие в программе?

Форма заявки на участие в программе собирает информацию, которую вы хотите получить от людей, подающих заявку в [вашу программу](/help/managing-a-program/dub-partners). Используйте её, чтобы спросить о веб-сайтах, аккаунтах в социальных сетях, опыте, размере аудитории или любых деталях, которые помогут вам решить, кого принять.

<ImageLink
  alt="Branded program application form"
  src="https://assets.dub.co/cms/program-application-form-1.png"
  href="https://partners.revroute.ru/dub/apply"
  cta="View example ↗"
/>

<Tip>
  В Revroute вы можете создать пользовательскую форму заявки для каждой [группы
  партнеров](/help/managing-a-program/getting-started/partner-groups) в вашей программе. Это полезно для
  сбора различной информации от разных типов партнеров — например, инфлюенсеров
  vs аффилиатов vs реселлеров.

По умолчанию при создании новой группы партнеров используются детали формы
заявки из группы партнеров по умолчанию, но вы всегда можете отредактировать
их позже.

</Tip>

## Где найти конструктор формы заявки?

На странице **Groups** откройте [группу партнеров](/help/managing-a-program/getting-started/partner-groups), для которой вы хотите настроить форму заявки.

![Выбор группы](https://assets.dub.co/cms/program-group-click.png)

В навигации по вкладкам нажмите **Branding**, затем на панели содержимого выберите **Application form**, чтобы открыть редактор.

![Страница брендинга в панели управления](https://assets.dub.co/cms/program-branding-tab-1.png)

Для удобства, [вот быстрая ссылка](https://app.dub.co/program/groups/default/branding?tab=application), которая перенесет вас непосредственно к конструктору формы заявки для группы по умолчанию вашей программы.

## Обязательные поля

Имя, email и страна всегда обязательны и отображаются в верхней части каждой формы. Вы не можете удалить или переименовать их.

![Обязательные поля](https://assets.dub.co/cms/application-mandatory.png)

## Редактирование и добавление полей

Наведите курсор на содержимое формы в конструкторе, чтобы открыть действия для добавления, редактирования, удаления и изменения порядка полей. Содержимое, отображаемое над формой, можно только редактировать. Это обеспечивает единообразие макета страницы для заявителей.

![Страница бренда в панели управления](https://assets.dub.co/cms/application-edit1.png)

## Типы полей ввода

Нажмите **Insert field**, чтобы открыть селектор, затем выберите тип ввода, который хотите добавить.

![Вставка поля](https://assets.dub.co/cms/application-insertfield2.png)

### Короткий текст

Используется для коротких ответов, таких как имена, однострочные описания или URL.

![Добавление полей короткого текста](https://assets.dub.co/cms/application-shorttext.png)

| Параметр          | Детали                                | Пример                              |
| ----------------- | ------------------------------------- | ----------------------------------- |
| Input label       | Текст над полем                       | _"Link to your top social profile"_ |
| Input placeholder | Подсказка внутри пустого поля         | _"x.com/steventey"_                 |
| Required          | Сделать поле обязательным             | Yes                                 |
| Max characters    | Установить лимит символов при необходимости | 120                            |

### Длинный текст

Используется для развернутых ответов, требующих нескольких строк.

![Добавление полей длинного текста](https://assets.dub.co/cms/application-longtext.png)

| Параметр          | Детали                                | Пример                                                      |
| ----------------- | ------------------------------------- | ----------------------------------------------------------- |
| Input label       | Текст над полем                       | "Tell us about your content strategy"                       |
| Input placeholder | Подсказка внутри пустого поля         | "Describe your approach to creating and sharing content..." |
| Required          | Сделать поле обязательным             | Yes                                                         |
| Max characters    | Установить лимит символов при необходимости | 500                                                    |

### Выпадающий список

Лучше всего подходит, когда у вас более пяти вариантов. Вы можете добавлять, переупорядочивать и удалять варианты в любое время.

![Добавление выпадающего списка](https://assets.dub.co/cms/application-dropdown.png)

| Параметр                  | Детали                                    | Пример                                                      |
| ------------------------- | ----------------------------------------- | ----------------------------------------------------------- |
| Input label               | Текст над полем                           | "Which platforms do you use?"                               |
| Required                  | Сделать поле обязательным                 | No                                                          |
| Allow multiple selections | Переключить для разрешения нескольких вариантов | Yes (for checkboxes)                                   |
| Options                   | Варианты, которые видит заявитель          | "Instagram", "TikTok", "YouTube", "LinkedIn", "X (Twitter)" |

### Множественный выбор

Поле множественного выбора может служить нескольким целям:

1. **Radio** — список элементов, из которого заявитель может выбрать только один
2. **Checkbox** — список элементов, из которого заявитель может выбрать один или несколько

По умолчанию начальный ввод установлен как radio, но может адаптироваться к вашим потребностям.

![Добавление множественного выбора](https://assets.dub.co/cms/application-multiplechoice.png)

| Параметр    | Детали                         | Пример                               |
| ----------- | ------------------------------ | ------------------------------------ |
| Input label | Текст над полем                | "What's your primary content focus?" |
| Required    | Сделать поле обязательным      | Yes                                  |
| Options     | Варианты, которые видит заявитель | "Tech & SaaS", "Business & Finance" |

### Загрузка изображений

Добавьте поле для загрузки изображений в вашу заявку, чтобы партнеры могли поделиться доказательствами, примерами или вспомогательными материалами.

![Добавление загрузки изображений](https://assets.dub.co/cms/application-imageupload.png)

| Параметр              | Детали                                                                | Пример                  |
| --------------------- | --------------------------------------------------------------------- | ----------------------- |
| Input label           | Текст над полем                                                       | "Analytics screenshots" |
| Allow multiple images | Разрешить партнерам загружать несколько изображений. Минимум 2, максимум 10. |                    |
| Required              | Сделать поле обязательным                                             |                         |

### Веб-сайт и социальные сети

Запросите конкретные веб-сайты или аккаунты в социальных сетях. Включайте или отключайте каждую сеть и отмечайте обязательные. Например, вы можете сделать Instagram обязательным, а X — необязательным.

![Добавление полей веб-сайта и социальных сетей](https://assets.dub.co/cms/application-website.png)

<Tip>
  Вы можете добавить только один раздел веб-сайта и социальных сетей на форму.
  Если один уже присутствует, тип ввода не будет отображаться в меню Insert
  field.
</Tip>

## Публикация изменений

Когда вы будете готовы, нажмите Publish в правом верхнем углу, чтобы опубликовать ваши изменения.

![Кнопка публикации](https://assets.dub.co/cms/application-publish.png)
