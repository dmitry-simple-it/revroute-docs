## Создание собственных интеграций

Вы можете создавать собственные интеграции с инфраструктурой ссылок Revroute, используя наш [API](/docs/api-reference/introduction).

1. Ознакомьтесь с документацией по [созданию ссылок](/docs/api-reference/links/create) или [отслеживанию конверсий продаж](/docs/api-reference/track/sale).
2. Узнайте, как [интегрировать Revroute в ваше приложение](/docs/integrations/quickstart).
3. [Свяжитесь с нами](https://revroute.ru//support), чтобы разместить вашу интеграцию в маркетплейсе интеграций.

## Вебхуки интеграций

Revroute также поддерживает вебхуки для интеграций. Подробнее о них можно узнать в документации по [вебхукам](/docs/webhooks/introduction).

Вот список вебхуков, которые поддерживает Revroute:

- [`link.created`](/docs/webhooks/events/link-created) — при создании новой ссылки
- [`link.updated`](/docs/webhooks/events/link-updated) — при обновлении ссылки
- [`link.deleted`](/docs/webhooks/events/link-deleted) — при удалении ссылки
- [`link.clicked`](/docs/webhooks/events/link-clicked) — при клике по ссылке
- [`lead.created`](/docs/webhooks/events/lead-created) — при создании нового лида
- [`sale.created`](/docs/webhooks/events/sale-created) — при создании новой продажи
- [`partner.enrolled`](/docs/webhooks/events/partner-enrolled) — при регистрации нового партнёра в вашей программе
