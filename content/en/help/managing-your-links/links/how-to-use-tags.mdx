Tags are a great way to organize your links on Revroute. You can use tags to group links together and make it easier to find them later.

![Using tags to organize links on Revroute](https://assets.dub.co/changelog/new-tag-creation-flow.png)

## What is a Tag?

A tag is a label that you can assign to a link. Tags are color-coded and can be created and assigned to a link in just a few clicks.

With tags, you can:

1. Organize your links by campaigns, clients, or any other categories you can think of.
2. Filter your links by tags and get a shareable link to the filtered results.
3. [Filter your analytics by tags](/help/managing-your-links/analytics/filtering-analytics/filter-analytics-by-tags) to see how your campaigns are performing.

## How to create a Tag

To create a tag, navigate to the [Tags settings page](https://app.revroute.ru/settings/library/tags) in your Revroute workspace.

![Tags settings page](https://assets.dub.co/help/tags-settings-page.png)

Click on **Create Tag** to open up the tag creation modal, where you can create a new tag and choose a color for it:

![Tag creation modal](https://assets.dub.co/help/create-tag-modal.png)

You can also create new tags directly inside the [Revroute link builder](/help/managing-your-links/additional-guides/dub-link-builder). Once you've [created a link](/help/managing-your-links/links/how-to-create-link), click on the tag input field and type in the name of the tag you want to create.

![Input the name of the tag you want to create (e.g. ](https://assets.dub.co/help/tag-dropdown.png)

If the tag already exists, you can select it from the dropdown. Otherwise, press `Enter ↩` to create a new tag.

Once the tag is created, click on the "Create Link" button to assign the tag to the link.

## How to edit or delete a Tag

Tags can be edited via the [Tags settings page](https://app.revroute.ru/settings/library/tags) in your Revroute workspace.

![Editing tags in the Tags settings page](https://assets.dub.co/help/tags-settings-page-edit.png)

Simply click on the `⋮` button to open the menu popover for the tag, which gives you the following options:

- Edit the tag
- Copy the tag's unique ID (for [programmatic API use](/help/managing-your-links/links/how-to-use-tags#managing-tags-programmatically))
- Delete the tag

![Edit a tag by clicking on the ⋮ icon](https://assets.dub.co/help/tag-menu-dropdown.png)

You can also use the following keyboard shortcuts to manage your tags:

| Keyboard Shortcut | Action                   |
| ----------------- | ------------------------ |
| `E`               | Edit a tag               |
| `I`               | Copy the tag's unique ID |
| `X`               | Delete a tag             |

To open the tag editing modal, click on the "Edit" button. Inside the tag editing modal, you can:

- Rename the tag
- Change the color of the tag

![Tag editing modal](https://assets.dub.co/help/edit-tag-modal.png)

## Can I assign multiple tags to a link?

Yes you can. In the tags section for a given link, you can select multiple tags to assign to the link.

![Assign multiple tags to a link](https://assets.dub.co/help/multiple-tags-dropdown.png)

The tags will show up in the link's details, and you can filter your links by these tags.

![Multiple tags assigned to a link](https://assets.dub.co/help/multiple-tags-link.png)

## How many tags can I create?

The number of tags you can create depends on your plan:

- **Free plan**: Up to 5 tags
- [**Pro plan**](https://revroute.ru/pricing): Up to 25 tags
- [**Business plan**](https://revroute.ru/pricing): Unlimited tags
- [**Enterprise plan**](https://revroute.ru/): Unlimited tags

Check out our [pricing page](https://revroute.ru/pricing) for more information on our plans.
