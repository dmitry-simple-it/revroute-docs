Короткие ссылки — основа Dub. С платформой управления ссылками Dub вы можете создавать короткие ссылки и [QR-коды](/help/managing-your-links/links/custom-qr-codes) для маркетинговых кампаний и получать [аналитику мирового уровня](/help/managing-your-links/analytics/dub-analytics) по ним.

В центре Dub находится конструктор ссылок. Здесь вы найдёте все инструменты и функции, необходимые для создания короткой ссылки в Dub.

![Конструктор ссылок Dub](https://assets.dub.co/help/dub-link-builder.jpg)

## Возможности конструктора ссылок Dub

В левой половине конструктора ссылок Dub предлагает множество расширенных функций для создания и управления короткими ссылками.

1. **Выбор тегов**: Организуйте ссылки с помощью тегов для удобного поиска. [Подробнее](/help/managing-your-links/links/how-to-use-tags).
2. **Пользовательские превью ссылок**: Настройте отображение ссылки в социальных сетях с помощью ИИ для повышения конверсии. [Подробнее](/help/article/custom-link-previews).
3. **UTM-конструктор**: Добавляйте UTM-параметры к ссылкам. [Подробнее](/help/managing-your-links/links/utm-builder).
4. **Маскировка ссылок**: Скрывайте URL назначения за короткой ссылкой. [Подробнее](/help/managing-your-links/links/link-cloaking).
5. **Защита паролем**: Защищайте ссылки паролем. [Подробнее](/help/managing-your-links/links/password-protected-links).
6. **Срок действия**: Устанавливайте дату истечения срока действия ссылок. [Подробнее](/help/managing-your-links/links/link-expiration).
7. **Таргетинг по устройствам**: Задавайте отдельный URL назначения для iOS и Android. [Подробнее](/help/managing-your-links/links/device-targeting).
8. **Геотаргетинг**: Перенаправляйте пользователей на разные ссылки в зависимости от их местоположения. [Подробнее](/help/managing-your-links/links/geo-targeting).
9. **Индексация поисковыми системами**: Управляйте индексацией ссылок поисковыми системами. [Подробнее](/help/managing-your-links/additional-guides/how-noindex-works).
10. **Комментарии**: Добавляйте комментарии к ссылкам — для вас и вашей команды. [Подробнее](/help/managing-your-links/links/link-comments).

В правой половине конструктора ссылок вы видите предпросмотр в реальном времени того, как ваша короткая ссылка будет выглядеть на платформах социальных сетей: **X (ранее Twitter)**, **Facebook** и **LinkedIn**.

## Горячие клавиши в конструкторе ссылок Dub

Конструктор ссылок Dub также содержит горячие клавиши для ускорения создания ссылок. Вот список всех доступных горячих клавиш:

| Клавиша  | Действие                                                               |
| -------- | ---------------------------------------------------------------------- |
| `D`      | Открыть выбор домена                                                   |
| `T`      | Открыть выбор тегов                                                    |
| `Q`      | Открыть модальное окно QR-кода                                         |
| `L`      | Открыть модальное окно [превью ссылки](/help/article/custom-link-previews) |
| `U`      | Открыть [UTM-конструктор](/help/managing-your-links/links/utm-builder)                   |
| `E`      | Открыть модальное окно [срока действия](/help/managing-your-links/links/link-expiration) |
| `G`      | Открыть модальное окно [гео/таргетинга по устройствам](/help/managing-your-links/links/geo-targeting) |
| `P`      | Открыть модальное окно [пароля ссылки](/help/managing-your-links/links/password-protected-links) |
| `K`      | Переключить [маскировку ссылок](/help/managing-your-links/links/link-cloaking)           |
| `S`      | Переключить [индексацию поисковыми системами](/help/managing-your-links/additional-guides/how-noindex-works) |
| `A`      | Открыть модальное окно дополнительных настроек                         |

## Как открыть конструктор ссылок Dub

Есть 3 способа открыть конструктор ссылок Dub:

- 🐢 **Самый медленный**: Нажмите кнопку «Create link» на панели управления.
- 🐇 **Быстрее**: Используйте горячую клавишу `c`, чтобы открыть конструктор ссылок.
- 🐅 **Самый быстрый**: Просто вставьте валидный URL на панели управления. Это автоматически откроет конструктор ссылок и заполнит поле URL.

## Быстрые способы создания ссылок в Dub
