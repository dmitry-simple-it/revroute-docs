# Отслеживание кликов (client-side)

Click tracking нужен, когда вы хотите отслеживать клики по партнёрским/реферальным ссылкам **на клиенте**, не полагаясь на редиректы.

## Как работает

1. Скрипт “слушает” query‑параметры (например, `via`)
2. Сохраняет значение в cookie/хранилище
3. Использует его для последующей атрибуции событий

## Настройка

При необходимости задайте параметры, которые скрипт будет читать:

```html
<script
  defer
  src="https://app.revroute.ru/analytics/script.js"
  data-query-params='["via"]'
></script>
```
