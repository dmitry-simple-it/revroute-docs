# Reverse proxy

Если у вас строгие требования к безопасности/приватности, вы можете отдавать клиентский скрипт RevRoute через свой домен с помощью reverse proxy.

## Пример (идея)

1. Настройте прокси‑роут на вашем домене, который проксирует запрос к:
   - `https://app.revroute.ru/analytics/script.js` (и/или нужные варианты)
2. Подключайте скрипт уже с вашего домена:

```html
<script defer src="https://yourdomain.com/analytics/script.js"></script>
```

Важно: убедитесь, что заголовки кеширования и CORS настроены корректно для вашего сценария.
