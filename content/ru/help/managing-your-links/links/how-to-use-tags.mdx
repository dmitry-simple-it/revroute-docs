Теги - отличный способ организации ссылок в Dub. Используйте теги для группировки ссылок и быстрого поиска.

![Использование тегов для организации ссылок в Dub](https://assets.dub.co/changelog/new-tag-creation-flow.png)

## Что такое тег?

Тег - это метка, которую можно присвоить ссылке. Теги имеют цветовую кодировку и создаются в несколько кликов.

С помощью тегов вы можете:

1. Организовать ссылки по кампаниям, клиентам или любым другим категориям.
2. Фильтровать ссылки по тегам и получать ссылку на отфильтрованные результаты.
3. [Фильтровать аналитику по тегам](/help/managing-your-links/analytics/filtering-analytics/filter-analytics-by-tags) для отслеживания эффективности кампаний.

## Как создать тег

Перейдите на [страницу настроек тегов](https://app.dub.co/settings/library/tags) в вашем рабочем пространстве Dub.

![Страница настроек тегов](https://assets.dub.co/help/tags-settings-page.png)

Нажмите **Create Tag** для открытия модального окна создания тега, где можно задать имя и выбрать цвет:

![Модальное окно создания тега](https://assets.dub.co/help/create-tag-modal.png)

Также можно создавать теги прямо в [конструкторе ссылок Dub](/help/managing-your-links/additional-guides/dub-link-builder). После [создания ссылки](/help/managing-your-links/links/how-to-create-link) нажмите на поле ввода тега и введите название.

![Введите название тега](https://assets.dub.co/help/tag-dropdown.png)

Если тег уже существует, выберите его из выпадающего списка. Иначе нажмите `Enter` для создания нового.

После создания тега нажмите «Create Link» для присвоения тега ссылке.

## Как отредактировать или удалить тег

Теги редактируются на [странице настроек тегов](https://app.dub.co/settings/library/tags).

![Редактирование тегов](https://assets.dub.co/help/tags-settings-page-edit.png)

Нажмите кнопку `⋮` для открытия меню с опциями:

- Редактировать тег
- Скопировать уникальный ID тега (для [программного использования](/help/managing-your-links/links/how-to-use-tags#managing-tags-programmatically))
- Удалить тег

![Меню тега](https://assets.dub.co/help/tag-menu-dropdown.png)

Горячие клавиши для управления тегами:

| Сочетание | Действие              |
| --------- | --------------------- |
| `E`       | Редактировать тег     |
| `I`       | Скопировать ID тега   |
| `X`       | Удалить тег           |

В модальном окне редактирования можно:

- Переименовать тег
- Изменить цвет тега

![Модальное окно редактирования](https://assets.dub.co/help/edit-tag-modal.png)

## Можно ли присвоить несколько тегов ссылке?

Да. В разделе тегов для ссылки можно выбрать несколько тегов.

![Несколько тегов для ссылки](https://assets.dub.co/help/multiple-tags-dropdown.png)

Теги отображаются в деталях ссылки, и вы можете фильтровать ссылки по ним.

![Несколько тегов на ссылке](https://assets.dub.co/help/multiple-tags-link.png)

## Сколько тегов можно создать?

Количество зависит от тарифа:

- **Free**: До 5 тегов
- [**Pro**](https://dub.co/pricing): До 25 тегов
- [**Business**](https://dub.co/pricing): Без ограничений
- [**Enterprise**](https://dub.co/enterprise): Без ограничений

Подробнее на [странице тарифов](https://dub.co/pricing).
