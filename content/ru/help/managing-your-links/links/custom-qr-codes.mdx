Как freemium-продукт, мы полагаемся на сарафанное радио для распространения информации о Revroute. Поэтому на бесплатном тарифе Revroute вы можете использовать только логотип QR-кода по умолчанию для ваших коротких ссылок.

![Логотип QR-кода по умолчанию в Revroute](https://assets.dub.co/help/free-plan-qrcode.png)

Если вы хотите удалить логотип Revroute или загрузить свой собственный, пожалуйста, рассмотрите [переход на тариф Pro](https://revroute.ru/pricing#how-to-upgrade-to-pro) для поддержки нас. Спасибо!

## Как установить пользовательский логотип QR-кода?

После перехода на тариф Pro вы можете установить пользовательский QR-код для ваших коротких ссылок.

Есть два способа установить пользовательский логотип QR-кода:

- [Для всего рабочего пространства](#установка-пользовательского-логотипа-qr-кода-для-всего-рабочего-пространства) — применяется ко всем коротким ссылкам в вашем рабочем пространстве
- [Для конкретного домена](#установка-пользовательского-логотипа-qr-кода-для-конкретного-домена) — применяется ко всем коротким ссылкам в этом домене

<Tip>
  Если пользовательский логотип QR-кода установлен как для рабочего
  пространства, так и для домена, логотип QR-кода для домена будет иметь
  приоритет.
</Tip>

### Вариант 1: Установка пользовательского логотипа QR-кода для всего рабочего пространства

Чтобы установить пользовательский логотип QR-кода для всего рабочего пространства, сначала необходимо установить логотип рабочего пространства, выполнив следующие шаги:

1. Перейдите на [страницу настроек рабочего пространства](https://app.revroute.ru/settings), нажав на вкладку **Settings** в боковой панели.
2. Прокрутите до секции **Workspace Logo** и загрузите логотип вашей компании. Для лучших результатов мы рекомендуем использовать квадратное изображение.

![Секция логотипа рабочего пространства](https://assets.dub.co/help/workspace-logo-section.png)

После установки логотипа рабочего пространства ваш QR-код автоматически обновится и будет использовать логотип вашего рабочего пространства.

![Пользовательский логотип QR-кода](https://assets.dub.co/changelog/custom-qr-codes.png)

### Вариант 2: Установка пользовательского логотипа QR-кода для конкретного домена

Если вы управляете несколькими брендами/продуктами в Revroute, вы можете установить пользовательские логотипы QR-кодов для каждого из них для лучшей узнаваемости бренда.

![Установка пользовательских логотипов QR-кодов для ваших доменов](https://assets.dub.co/changelog/custom-qr-code-logo.jpg)

Чтобы установить пользовательский логотип QR-кода для конкретного домена, выполните следующие шаги:

1. Перейдите на [вкладку Domains настроек рабочего пространства](https://app.revroute.ru/domains).
2. Нажмите кнопку `⋮` рядом с доменом, для которого хотите установить пользовательский логотип QR-кода.
3. Выберите **Edit Domain** для редактирования настроек домена.
4. Активируйте переключатель **Custom QR code logo**.
5. Загрузите пользовательский логотип QR-кода. Для лучших результатов мы рекомендуем использовать квадратное изображение.
6. Нажмите кнопку **Save changes** для сохранения изменений.

![Установка пользовательского логотипа QR-кода для пользовательского домена](https://assets.dub.co/help/domain-custom-qr-code-logo.png)

## Программное создание QR-кодов

После установки пользовательского логотипа QR-кода вы можете использовать наш [API QR-кодов](/docs/api-reference/endpoint/retrieve-a-qr-code) для программного создания QR-кодов.

На тарифах Pro и выше вы можете выбрать, включать ваш логотип в сгенерированные QR-коды или скрывать его.

- Если для вашего рабочего пространства установлен логотип, ваши QR-коды автоматически будут содержать этот логотип, но вы можете скрыть его с помощью параметра `hideLogo` ([пример](https://api.dub.co/qr?url=https://d.to/try?qr=1))
- Если для вашего рабочего пространства не установлен логотип, QR-коды будут без логотипа.
- Также вы можете передать пользовательский логотип в QR-коды с помощью параметра `logo` ([пример](https://api.dub.co/qr?url=https://d.to/try?qr=1&logo=https://assets.dub.co/wordmark-square.png)).
