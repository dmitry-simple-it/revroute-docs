<Tip>
  Эта функция доступна только на [тарифах Pro и
  выше](https://dub.co/pricing).
</Tip>

В Dub вы можете установить срок действия для ссылок, нажав кнопку **Expiration** в конструкторе ссылок. Откроется модальное окно **Link Expiration**.

Быстрый доступ — сочетание клавиш `E` в конструкторе ссылок.

Заполните следующие поля:

1. **Date and Time**: Дата и время истечения срока действия ссылки.
2. **Expiration URL**: URL, на который пользователь будет перенаправлен после истечения срока.

Нажмите **Add expiration** для сохранения.

Можно использовать естественный язык для ввода даты и времени. Например, «in 2 hours» или «tomorrow at 5PM» — Dub автоматически преобразует ввод в правильный формат.

Также можно выбрать дату и время вручную через выпадающий календарь.

Если установлен срок действия, после его истечения пользователи увидят страницу «Expired Link» ([пример](https://dub.sh/expired)).

![Страница истёкшей ссылки](https://assets.dub.co/cms/expired-link-placeholder.png)

<Warning>
  Примечание: клики по ссылке после истечения срока ***не*** будут
  отслеживаться.
</Warning>

## Пользовательский URL для истёкших ссылок

Вы можете задать собственный URL перенаправления для истёкших ссылок. Это полезно, если хотите направлять пользователей на конкретную страницу.

Введите URL в поле **Expiration URL** в разделе **Link Expiration**.

![URL для истёкшей ссылки](https://assets.dub.co/help/link-expiration-url.png)

## URL по умолчанию для всех ссылок домена

Также можно задать URL по умолчанию для истёкших ссылок на всём домене. Это полезно для перенаправления пользователей при истечении _любой ссылки на домене_.

Как настроить:

1. Перейдите на [страницу настроек доменов](https://app.dub.co/domains) в дашборде Dub.
2. Нажмите кнопку `⋮` у нужного домена — откроется выпадающее меню.

![Всплывающее меню](https://assets.dub.co/help/domain-popover-menu.png)

4. Выберите **Edit Domain** — откроется модальное окно.
5. В поле **Default Expiration URL** введите URL перенаправления.

![URL по умолчанию для домена](https://assets.dub.co/help/domain-expiration-url.png)

7. Нажмите **Save changes** для сохранения.
