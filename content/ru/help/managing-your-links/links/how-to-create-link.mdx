Конструктор ссылок Dub включает мощные функции: [UTM-конструктор](/help/managing-your-links/links/utm-builder), [таргетинг по устройствам](/help/managing-your-links/links/device-targeting), [защита паролем](/help/managing-your-links/links/password-protected-links), [срок действия](/help/managing-your-links/links/link-expiration) и другие.

Dub — единственная платформа управления ссылками, позволяющая [настраивать карточки соцсетей](/help/article/custom-link-previews) для ваших ссылок и полностью контролировать их внешний вид при публикации.

В этой статье мы рассмотрим, как использовать эти функции для создания первой короткой ссылки в Dub.

<Steps>
  <Step title="Откройте конструктор ссылок Dub">

    Есть 3 способа открыть [конструктор ссылок Dub](/help/managing-your-links/additional-guides/dub-link-builder):

    1. **Медленный**: Нажмите кнопку «Create link» на дашборде.
    2. **Быстрый**: Используйте сочетание клавиш `c`.
    3. **Самый быстрый**: Вставьте валидный URL на дашборде — конструктор откроется автоматически с заполненным полем URL.

    ![Конструктор ссылок Dub](https://assets.dub.co/help/dub-link-builder.jpg)

    Конструктор также показывает превью вашей короткой ссылки в соцсетях: X (ранее Twitter), Facebook и LinkedIn.

  </Step>
  <Step title="Введите URL назначения">
    Введите URL назначения в поле «Destination URL». Можно вводить URL с протоколом `https://` или без — Dub автоматически отформатирует его правильно.

  </Step>

  <Step title="Задайте слаг короткой ссылки" optional>
    В поле **Short Link** вы можете:
    1. Ввести собственный слаг
    2. Сгенерировать случайный слаг
    3. [Сгенерировать слаг с помощью AI](https://dub.co/blog/introducing-dub-ai)

    <video src="https://assets.dub.co/help/generate-shortlink-ai.mp4"
        loop
        autoPlay
        muted
        playsInline
      ></video>

    <Tip>
      Этот шаг необязателен. Если не задать слаг, Dub сгенерирует его автоматически.
    </Tip>

  </Step>

  <Step title="Настройте превью ссылки" optional>
    Вы также можете [настроить превью ссылки](/help/article/custom-link-previews) для контроля внешнего вида при публикации в X (ранее Twitter), LinkedIn, Slack, Telegram, iMessage и других платформах.

    ![Создание пользовательских превью для ссылок](https://assets.dub.co/help/custom-link-previews.png)

    Можно использовать AI для генерации превью. [Подробнее](/help/article/custom-link-previews).

    <video src="https://assets.dub.co/help/generate-metatags-ai.mp4"
        loop
        autoPlay
        muted
        playsInline
      ></video>

  </Step>

  <Step title="Добавьте теги к ссылке" optional>
    При желании добавьте теги для организации ссылок и удобного поиска.

    ![К одной ссылке Dub можно привязать несколько тегов](https://assets.dub.co/help/multiple-tags-dropdown.png)

    Вы также можете [фильтровать аналитику по тегам](/help/managing-your-links/analytics/filtering-analytics/filter-analytics-by-tags) для создания отчётов по кампаниям.

  </Step>
</Steps>

Готово! Вы создали свою первую короткую ссылку в Dub.

Dub также предлагает расширенные функции для ссылок:

1. **UTM-конструктор**: Добавление UTM-параметров. [Подробнее](/help/managing-your-links/links/utm-builder).
2. **Маскировка ссылок**: Скрытие URL назначения за короткой ссылкой. [Подробнее](/help/managing-your-links/links/link-cloaking).
3. **Защита паролем**: Доступ по паролю. [Подробнее](/help/managing-your-links/links/password-protected-links).
4. **Срок действия**: Установка даты истечения. [Подробнее](/help/managing-your-links/links/link-expiration).
5. **Таргетинг по устройствам**: Разные URL для iOS и Android. [Подробнее](/help/managing-your-links/links/device-targeting).
6. **Гео-таргетинг**: Редирект по местоположению пользователя. [Подробнее](/help/managing-your-links/links/geo-targeting).
7. **Индексация**: Управление индексацией поисковиками. [Подробнее](/help/managing-your-links/additional-guides/how-noindex-works).
8. **Комментарии**: Заметки к ссылкам для вас и команды. [Подробнее](/help/managing-your-links/links/link-comments).
