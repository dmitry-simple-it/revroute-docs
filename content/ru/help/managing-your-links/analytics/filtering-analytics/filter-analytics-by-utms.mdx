sidebarTitle: "По UTM-тегам"
---

В Revroute вы можете фильтровать аналитику по [UTM-параметрам](/help/managing-your-links/links/utm-builder). Это позволяет создавать мощные отчёты по кампаниям для ваших ссылок.

Например, если вы используете нашу [функцию шаблонов UTM](/help/managing-your-links/links/how-to-create-utm-templates) для создания стандартизированных UTM-параметров для вашей команды в маркетинговых кампаниях, вам нужен способ увидеть, как каждый из этих UTM работает на вкладке Analytics.

С помощью вкладки **UTM Parameters** вы можете легко отслеживать эффективность каждого UTM-параметра в реальном времени.

![](https://assets.dub.co/cms/utm-analytics.jpg)

Вы также можете отфильтровать аналитику по конкретному UTM-параметру, для которого хотите создать пользовательский отчёт:

<Steps>

     <Step title="Откройте выпадающий фильтр аналитики">
        На вкладке [Analytics](https://app.revroute.ru/analytics) вашего рабочего пространства Revroute нажмите на кнопку **Filter**, чтобы открыть выпадающий фильтр:

        ![Выпадающий фильтр аналитики](https://assets.dub.co/cms/analytics-filter-dropdown-utm.png)

        <Tip>
          Вы также можете использовать сочетание клавиш `F` для быстрого открытия выпадающего фильтра.
        </Tip>

    </Step>
    <Step title="Выберите фильтр 'UTM Source'">
        Выберите опцию **UTM Source** в выпадающем меню, чтобы открыть фильтр по источникам:

        ![Выпадающий список UTM Source в аналитике](https://assets.dub.co/cms/analytics-utm-source-dropdown.png)

        Если вы хотите фильтровать по другим UTM-параметрам, вы можете выбрать их соответствующим образом в основном выпадающем фильтре.

    </Step>
    <Step title="Нажмите на UTM-источник, по которому хотите фильтровать">
        Наконец, нажмите на UTM-источник, по которому хотите фильтровать:

        ![Отфильтрованная аналитика по UTM-источнику](https://assets.dub.co/cms/filtered-analytics-by-utm.png)
    </Step>

</Steps>
