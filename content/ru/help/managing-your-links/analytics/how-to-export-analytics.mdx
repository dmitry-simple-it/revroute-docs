Аналитика может быть мощным инструментом для понимания того, как ваша аудитория взаимодействует с контентом, и для принятия решений на основе данных.

Для Dub всегда было приоритетом предоставлять [аналитику мирового класса](/help/managing-your-links/analytics/dub-analytics) для всех.

![CSV-файлы универсальны и могут быть открыты в различных инструментах](https://assets.dub.co/help/import-zip-bi.png)

Чтобы получить больше от вашей аналитики, вы можете экспортировать данные аналитики в CSV-файл. Это может быть полезно, если вы хотите проанализировать данные в инструменте бизнес-аналитики (BI), поделиться ими с командой или импортировать их в электронную таблицу.

## Экспорт данных аналитики ссылок

Чтобы экспортировать данные аналитики ссылок, выполните следующие шаги:

<Steps>

<Step title="Перейдите на страницу аналитики">

Перейдите в [панель управления Dub](https://d.to/register), войдите в систему и выберите проект, из которого хотите экспортировать данные.

Нажмите на вкладку **Analytics** в верхней панели навигации.

</Step>

<Step title="Скачайте данные">

Нажмите на кнопку `⋮` рядом с [выбором диапазона дат](https://dub.co/changelog/date-range-picker), чтобы открыть выпадающее меню дополнительных параметров.

Нажмите на опцию **Download as CSV**.

![Нажмите на иконку загрузки для экспорта данных](https://assets.dub.co/help/download-csv-button.png)

Через несколько секунд zip-файл будет загружен на ваш компьютер. Он содержит CSV-файл для каждого типа аналитики, который вы уже знаете по панели управления (например, `countries.csv`, `devices.csv` и т. д.).

![Найдите CSV-файлы в загруженном zip-архиве](https://assets.dub.co/help/download-file.png)

</Step>

</Steps>
